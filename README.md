# Server Monitoring Bash Script

Простой bash-скрипт для базовой проверки состояния Linux-сервера.

## Возможности

* вывод информации о нагрузке системы;
* проверка использования CPU и RAM;
* просмотр активных процессов;
* получение краткой информации о состоянии сервера;
* запуск в Linux, WSL или на удалённом сервере.

## Запуск

```bash
chmod +x script.sh
./script.sh
```

Либо:

```bash
bash script.sh
```
